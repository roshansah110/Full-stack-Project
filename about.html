About Focus Tracker

Application Overview

Focus Tracker helps you set clear goals and visually track your progress toward completion.

Prioritize tasks using importance tags (High, Medium, Low) and monitor overall completion rate.

Technical Details

Built with: HTML5, CSS3, Vanilla JavaScript

Data Storage: Local Browser Storage (localStorage)

This application uses the Roboto typeface for a clean, modern aesthetic.
